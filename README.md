# tuq-web-app
Web app - Control de acceso GTU


# Descarga de proyecto GitHub y ejecución
### -Instala Git en tu máquina: 
https://git-scm.com/
<br/>
<br/>

### -Verifica con CMD o PowerShell que este instalado git
```
git -v
```
<br/>

### -Crear una nueva carpeta con el nombre “computo_nube”
<br/>


### -Abre Visual Code y accede a la carpeta creada “computo_nube”
Archivo -> Abrir folder -> Selecciona la carpeta “computo_nube”
<br/>
<br/>

### -Abre una nueva terminal en Visual Code
Terminal -> Nueva terminal
<br/>
<br/>

### -Clona el repositorio (crea una copia de forma local)
```
git clone https://github.com/xAlonsoGtu/tuq-web-app.git
```
<br/>

### -Accede al proyecto, si es que no estas dentro ya.
```
cd tuq-web-app
```
<br/>

### -Instala dependencias de nodeJs, se creará carpeta “node_modules"
```
npm install
```
<br/>

### -Ejecuta nodeJs
```
npm start
```
<br/>

### -Accede a la carpeta de react
```
cd spa
```
<br/>

### -Instala dependencias de react, se creará carpeta “node_modules"
```
npm install
```
<br/>

### -Ejecuta react
```
npm start
```
<br/>
